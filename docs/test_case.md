## 1. 테스트 개요

본 문서는 책첵(BookCheck) 웹앱의 기능 요구사항(FR-01 ~ FR-08)에 대한 수동 테스트 케이스를 정의한다.  
테스트 환경: Chrome 브라우저 / index.html 로컬 실행

---

## 2. 테스트 케이스 목록

### TC-01 | 책 추가 — 정상 입력

| 항목          | 내용                                                                       |
| ------------- | -------------------------------------------------------------------------- |
| 관련 요구사항 | FR-01                                                                      |
| 전제 조건     | 앱이 열려 있고 목록이 비어 있음                                            |
| 입력          | 제목: "채식주의자", 저자: "한강", 상태: 완독, 별점: 5, 메모: "강렬한 문체" |
| 기대 결과     | 목록에 해당 책이 추가되고 통계의 전체 권수가 1 증가함                      |
| 실제 결과     |                                                                            |
| 결과          | ⬜ 통과 / ⬜ 실패                                                          |
